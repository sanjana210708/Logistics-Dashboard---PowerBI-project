# Logistics Operations Dashboard – Power BI

## Project Description

This project presents an interactive Logistics Operations Dashboard built using Power BI to analyze and visualize key operational data of a logistics company. The dashboard provides insights into driver performance, vehicle utilization, and hub operations.

By transforming raw logistics data into visual insights, the dashboard helps stakeholders monitor operations, identify inefficiencies, and make data-driven decisions.

## Tools Used

* Power BI
* Power Query
* DAX
* CSV Dataset

## Key Insights

* Analysis of driver information including driver allocation and performance metrics.
* Hub-wise operational insights showing activity across different logistics hubs.
* Vehicle data analysis including vehicle availability and usage.
* Identification of patterns and trends in logistics operations through interactive visualizations.

## Project Files Explanation

* **dashboard/Logistics Dashboard.pbix** – Power BI dashboard file containing all visualizations and data model.
* **dataset/Logistics Dataset.csv** – Dataset used for building the dashboard.
* **screenshots/** – Images of the dashboard showing the key visualizations and insights.
